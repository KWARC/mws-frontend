41.2-5 OnTuples
‣ OnTuples( tup, g )( function )

Let tup be a list. OnTuples returns the list formed by the images of all points x 𝑥 x of tup via the action function OnPoints (41.2-1), applied to x 𝑥 x and g.

(OnSets (41.2-4) is an action on lists that additionally sorts the entries of the result.)
