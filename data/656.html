72.4-1 Characteristic
‣ Characteristic( chi )( attribute )

The characteristic of class functions is zero, as for all list of cyclotomics. For class functions of a p 𝑝 p -modular character table, such as Brauer characters, the prime p 𝑝 p is given by the UnderlyingCharacteristic (71.9-5) value of the character table.

gap> Characteristic( irr[1] );
0
gap> irrmod2:= Irr( g, 2 );
[ Character( BrauerTable( Sym( [ 1 .. 4 ] ), 2 ), [ 1, 1 ] ), 
  Character( BrauerTable( Sym( [ 1 .. 4 ] ), 2 ), [ 2, -1 ] ) ]
gap> Characteristic( irrmod2[1] );
0
gap> UnderlyingCharacteristic( UnderlyingCharacterTable( irrmod2[1] ) );
2
