56.5-2 Units
‣ Units( R )( attribute )

Units returns the group of units of the ring R. This may either be returned as a list or as a group.

An element r 𝑟 r is called a unit of a ring R 𝑅 R if r 𝑟 r has an inverse in R 𝑅 R . It is easy to see that the set of units forms a multiplicative group.

gap> Units( GaussianIntegers );
[ -1, 1, -E(4), E(4) ]
gap> Units( GF( 16 ) );
<group with 1 generators>
