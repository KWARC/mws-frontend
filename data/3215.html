41 Group Actions

A group action is a triple ( G , Ω , μ ) 𝐺 Ω 𝜇 (G,\Omega,\mu) , where G 𝐺 G is a group, Ω Ω \Omega a set and μ : Ω × G → Ω : 𝜇 → Ω 𝐺 Ω \mu\colon\Omega\times G\rightarrow\Omega a function that is compatible with the group arithmetic. We call Ω Ω \Omega the domain of the action.

In GAP, Ω Ω \Omega can be a duplicate-free collection (an object that permits access to its elements via the Ω ⁢ [ n ] Ω delimited-[] 𝑛 \Omega[n] operation, for example a list), it does not need to be sorted (see IsSet (21.17-4)).

The acting function μ 𝜇 \mu is a binary GAP function that returns the image μ ⁢ ( x , g ) 𝜇 𝑥 𝑔 \mu(x,g) for a point x ∈ Ω 𝑥 Ω x\in\Omega and a group element g ∈ G 𝑔 𝐺 g\in G .

In GAP, groups always act from the right, that is μ ⁢ ( μ ⁢ ( x , g ) , h ) = μ ⁢ ( x , g ⁢ h ) 𝜇 𝜇 𝑥 𝑔 ℎ 𝜇 𝑥 𝑔 ℎ \mu(\mu(x,g),h)=\mu(x,gh) .

GAP does not test whether the acting function μ 𝜇 \mu satisfies the conditions for a group operation but silently assumes that is does. (If it does not, results are unpredictable.)

The first section of this chapter, 41.1, describes the various ways how operations for group actions can be called.

Functions for several commonly used action are already built into GAP. These are listed in section 41.2.

The sections 41.7 and 41.8 describe homomorphisms and mappings associated to group actions as well as the permutation group image of an action.

The other sections then describe operations to compute orbits, stabilizers, as well as properties of actions.

Finally section 41.12 describes the concept of "external sets" which represent the concept of a G 𝐺 G -set and underly the actions mechanism.
