76.3-8 IsPackageMarkedForLoading
‣ IsPackageMarkedForLoading( name, version )( function )

This function can be used in the code of a package A 𝐴 A , say, for testing whether the package name in version version will be loaded after the LoadPackage (76.2-1) call for the package A 𝐴 A has been executed. This means that the package name had been loaded before, or has been (directly or indirectly) requested as a needed or suggested package of the package A 𝐴 A or of a package whose loading requested that A 𝐴 A was loaded.
