73.5-1 CompositionMaps
‣ CompositionMaps( paramap2, paramap1[, class] )( function )

The composition of two parametrized maps paramap1, paramap2 is defined as the parametrized map comp that contains all compositions f 2 ∘ f 1 subscript 𝑓 2 subscript 𝑓 1 f_{2}\circ f_{1} of elements f 1 subscript 𝑓 1 f_{1} of paramap1 and f 2 subscript 𝑓 2 f_{2} of paramap2. For example, the composition of a character χ 𝜒 \chi of a group G 𝐺 G by a parametrized class fusion map from a subgroup H 𝐻 H to G 𝐺 G is the parametrized map that contains all restrictions of χ 𝜒 \chi by elements of the parametrized fusion map.

CompositionMaps(paramap2, paramap1) is a parametrized map with entry CompositionMaps(paramap2, paramap1, class) at position class. If paramap1[class] is an integer then CompositionMaps(paramap2, paramap1, class) is equal to paramap2[ paramap1[ class ] ]. Otherwise it is the union of paramap2[i] for i in paramap1[ class ].

gap> map1:= [ 1, [ 2 .. 4 ], [ 4, 5 ], 1 ];;
gap> map2:= [ [ 1, 2 ], 2, 2, 3, 3 ];;
gap> CompositionMaps( map2, map1 );
[ [ 1, 2 ], [ 2, 3 ], 3, [ 1, 2 ] ]
gap> CompositionMaps( map1, map2 );
[ [ 1, 2, 3, 4 ], [ 2, 3, 4 ], [ 2, 3, 4 ], [ 4, 5 ], [ 4, 5 ] ]
