55.1-2 IsNearAdditiveMagmaWithZero
‣ IsNearAdditiveMagmaWithZero( obj )( category )

A near-additive magma-with-zero in GAP is a near-additive magma A 𝐴 A with an operation 0* (or Zero (31.10-3)) that yields the zero element of A 𝐴 A .

So a near-additive magma-with-zero A does always contain a unique additively neutral element z 𝑧 z , i.e., z + a = a = a + z 𝑧 𝑎 𝑎 𝑎 𝑧 z+a=a=a+z holds for all a ∈ A 𝑎 𝐴 a\in A (see AdditiveNeutralElement (55.3-5)). This zero element z 𝑧 z can be computed with the operation Zero (31.10-3), by applying this function to A 𝐴 A or to any element a 𝑎 a in A 𝐴 A . The zero element can be computed also as 0 * a 𝑎 a , for any a 𝑎 a in A 𝐴 A .

Note that it may happen that a near-additive magma containing a zero does not lie in the category IsNearAdditiveMagmaWithZero (see 31.6).
