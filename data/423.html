56.3-2 RingWithOne
‣ RingWithOne( r, s, ... )( function )
‣ RingWithOne( coll )( function )

In the first form RingWithOne returns the smallest ring with one that contains all the elements r, s, … … \ldots In the second form RingWithOne returns the smallest ring with one that contains all the elements in the collection C. If any element is not an element of a ring or if the elements lie in no common ring an error is raised.
