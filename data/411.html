56.4-8 IsJacobianRing
‣ IsJacobianRing( R )( property )

is true if the Jacobi identity holds in the ring R, and false otherwise. The Jacobi identity means that x * ( y * z ) + z * ( x * y ) + y * ( z * x ) 𝑥 𝑦 𝑧 𝑧 𝑥 𝑦 𝑦 𝑧 𝑥 x*(y*z)+z*(x*y)+y*(z*x) is the zero element of R, for all elements x 𝑥 x , y 𝑦 y , z 𝑧 z in R.

gap> L:= FullMatrixLieAlgebra( GF( 5 ), 7 );

true
