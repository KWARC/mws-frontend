51.8-12 IsRegularDClass
‣ IsRegularDClass( Dclass )( property )

returns true if the Greens D 𝐷 D class Dclass is regular. A D 𝐷 D class is regular if and only if each of its elements is regular, which in turn is true if and only if any one element of Dclass is regular. Idempotents are regular since e ⁢ e ⁢ e = e 𝑒 𝑒 𝑒 𝑒 eee=e so it follows that a Green's D 𝐷 D class containing an idempotent is regular. Conversely, it is true that a regular D 𝐷 D class must contain at least one idempotent. (See [How76, Prop. 3.2].)
