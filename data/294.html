22.3-1 UnionBlist
‣ UnionBlist( blist1, blist2[, ...] )( function )
‣ UnionBlist( list )( function )

In the first form UnionBlist returns the union of the boolean lists blist1, blist2, etc., which must have equal length. The union is a new boolean list that contains at position i 𝑖 i the value blist1 [ i ] delimited-[] 𝑖 [i] or blist2 [ i ] delimited-[] 𝑖 [i] or … … \ldots .

The second form takes the union of all blists (which as for the first form must have equal length) in the list list.
