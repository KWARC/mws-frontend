14.2-8 LogInt
‣ LogInt( n, base )( function )

LogInt returns the integer part of the logarithm of the positive integer n with respect to the positive integer base, i.e., the largest positive integer e 𝑒 e such that 𝑏𝑎𝑠𝑒 e ≤ 𝑛 superscript base 𝑒 n \textit{base}^{e}\leq\textit{n} . The function LogInt will signal an error if either n or base is not positive.

For base = 2 absent 2 =2 this is very efficient because the internal binary representation of the integer is used.

10
gap> 2^10;

gap> LogInt( 1, 10 );
