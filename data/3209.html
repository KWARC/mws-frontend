41.2-4 OnSets
‣ OnSets( set, g )( function )

Let set be a proper set (see 21.19). OnSets returns the proper set formed by the images of all points x 𝑥 x of set via the action function OnPoints (41.2-1), applied to x 𝑥 x and g.

OnSets is for example used to compute the action of a permutation group on blocks.

(OnTuples (41.2-5) is an action on lists that preserves the ordering of entries.)
