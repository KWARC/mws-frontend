62.9-6 IndicesOfAdjointBasis
‣ IndicesOfAdjointBasis( B )( attribute )

Let A be an algebra and let B be the basis that is output by AdjointBasis( Basis( A ) ). This function returns a list of indices. If i 𝑖 i is an index belonging to this list, then a ⁢ d ⁢ x i 𝑎 𝑑 subscript 𝑥 𝑖 adx_{i} is a basis vector of the matrix space spanned by a ⁢ d ⁢ A 𝑎 𝑑 𝐴 adA , where x i subscript 𝑥 𝑖 x_{i} is the i 𝑖 i -th basis vector of the basis B.

gap> L:= FullMatrixLieAlgebra( Rationals, 3 );;
gap> B:= AdjointBasis( Basis( L ) );;

[ 1, 2, 3, 4, 5, 6, 7, 8 ]
