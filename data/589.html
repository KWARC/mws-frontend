72.13-2 PermCharInfoRelative
‣ PermCharInfoRelative( tbl, tbl2, permchars )( function )

Let tbl and tbl2 be the ordinary character tables of two groups H 𝐻 H and G 𝐺 G , respectively, where H 𝐻 H is of index two in G 𝐺 G , and permchars either the permutation character ( 1 U ) G superscript subscript 1 𝑈 𝐺 (1_{U})^{G} , for a subgroup U 𝑈 U of G 𝐺 G , or a list of such permutation characters. PermCharInfoRelative returns a record with the same components as PermCharInfo (72.13-1), the only exception is that the entries of the ATLAS component are names relative to tbl.

More precisely, the i 𝑖 i -th entry of the ATLAS component is a string describing the decomposition of the i 𝑖 i -th entry in permchars. The degrees and distinguishing letters of the constituents refer to the irreducibles of tbl, as follows. The two irreducible characters of tbl2 of degree N 𝑁 N , say, that extend the irreducible character N 𝑁 N a of tbl are denoted by N 𝑁 N a + {}^{+} and N 𝑁 N a - {}^{-} . The irreducible character of tbl2 of degree 2 ⁢ N 2 𝑁 2N , say, whose restriction to tbl is the sum of the irreducible characters N 𝑁 N a and N 𝑁 N b is denoted as N 𝑁 N ab. Multiplicities larger than 1 1 1 of constituents are denoted by exponents.

(This format is useful mainly for multiplicity free permutation characters.)

gap> t:= CharacterTable( "A5" );;
gap> t2:= CharacterTable( "A5.2" );;
gap> List( Irr( t2 ), x -> x[1] );
[ 1, 1, 6, 4, 4, 5, 5 ]
gap> List( Irr( t ), x -> x[1] );
[ 1, 3, 3, 4, 5 ]
gap> permchars:= List( [ [1], [1,2], [1,7], [1,3,4,4,6,6,7] ],
>                      l -> Sum( Irr( t2 ){ l } ) );
[ Character( CharacterTable( "A5.2" ), [ 1, 1, 1, 1, 1, 1, 1 ] ), 
  Character( CharacterTable( "A5.2" ), [ 2, 2, 2, 2, 0, 0, 0 ] ), 
  Character( CharacterTable( "A5.2" ), [ 6, 2, 0, 1, 0, 2, 0 ] ), 
  Character( CharacterTable( "A5.2" ), [ 30, 2, 0, 0, 6, 0, 0 ] ) ]
gap> info:= PermCharInfoRelative( t, t2, permchars );;
gap> info.ATLAS;
[ "1a^+", "1a^{\\pm}", "1a^++5a^-", 
  "1a^++3ab+4(a^+)^{2}+5a^+a^{\\pm}" ]
