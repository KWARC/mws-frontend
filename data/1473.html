24.5-9 ReflectionMat
‣ ReflectionMat( coeffs[, conj][, root] )( function )

Let coeffs be a row vector. ReflectionMat returns the matrix of the reflection in this vector.

More precisely, if coeffs is the coefficients list of a vector v 𝑣 v w.r.t. a basis B 𝐵 B (see Basis (61.5-2)), say, then the returned matrix describes the reflection in v 𝑣 v w.r.t. B 𝐵 B as a map on a row space, with action from the right.

The optional argument root is a root of unity that determines the order of the reflection. The default is a reflection of order 2. For triflections one should choose a third root of unity etc. (see E (18.1-1)).

conj is a function of one argument that conjugates a ring element. The default is ComplexConjugate (18.5-2).

The matrix of the reflection in v 𝑣 v is defined as

\[ M = I_n + \overline{{v^{tr}}} \cdot (w-1) / ( v \overline{{v^{tr}}} ) \cdot v \]

where w 𝑤 w equals root, n 𝑛 n is the length of the coefficient list, and ¯ ¯ absent \overline{{\vphantom{x}}} denotes the conjugation.

So v 𝑣 v is mapped to w ⁢ v 𝑤 𝑣 wv , with default - v 𝑣 -v , and any vector x 𝑥 x with the property x ⁢ v t ⁢ r ¯ = 0 𝑥 ¯ superscript 𝑣 𝑡 𝑟 0 x\overline{{v^{tr}}}=0 is fixed by the reflection.
