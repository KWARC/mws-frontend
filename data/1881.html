16.2-4 Arrangements
‣ Arrangements( mset[, k] )( function )

returns the set of arrangements of the multiset mset that contain k elements. If k is not given it returns all arrangements of mset.

An arrangement of mset is an ordered selection without repetitions and is represented by a list that contains only elements from mset, but maybe in a different order.
