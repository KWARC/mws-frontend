62.4-7 IdentityFromSCTable
‣ IdentityFromSCTable( T )( function )

Let T be a structure constants table of an algebra A 𝐴 A of dimension n 𝑛 n . IdentityFromSCTable( T ) is either fail or the vector of length n 𝑛 n that contains the coefficients of the multiplicative identity of A 𝐴 A with respect to the basis that belongs to T.

gap> T:= EmptySCTable( 2, 0 );;
gap> SetEntrySCTable( T, 1, 1, [ 1, 1 ] );;
gap> SetEntrySCTable( T, 1, 2, [ 1, 2 ] );;
gap> SetEntrySCTable( T, 2, 1, [ 1, 2 ] );;
gap> IdentityFromSCTable( T );
[ 1, 0 ]
