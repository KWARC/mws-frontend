ErrorCount returns a count of the number of errors (including user interruptions) which have occurred in the GAP session so far. This count is reduced modulo 2 28 superscript 2 28 2^{28} on 32 32 32 bit systems, 2 60 superscript 2 60 2^{60} on 64 64 64 bit systems. The count is incremented by each error, even if GAP was started with the -T option to disable the break loop.
