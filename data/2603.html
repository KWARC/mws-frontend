61.9-7 IsSemiEchelonized
‣ IsSemiEchelonized( B )( property )

Let B be a basis of a Gaussian row or matrix space V 𝑉 V , say (see IsGaussianSpace (61.9-3)) over the field F 𝐹 F .

If V 𝑉 V is a row space then B is semi-echelonized if the matrix formed by its basis vectors has the property that the first nonzero element in each row is the identity of F 𝐹 F , and all values exactly below these pivot elements are the zero of F 𝐹 F (cf. SemiEchelonMat (24.10-1)).

If V 𝑉 V is a matrix space then B is semi-echelonized if the matrix obtained by replacing each basis vector by the concatenation of its rows is semi-echelonized (see above, cf. SemiEchelonMats (24.10-4)).

gap> V:= GF(2)^2;;
gap> B1:= Basis( V, [ [ 0, 1 ], [ 1, 0 ] ] * Z(2) );;
gap> IsSemiEchelonized( B1 );
true
gap> B2:= Basis( V, [ [ 0, 1 ], [ 1, 1 ] ] * Z(2) );;
gap> IsSemiEchelonized( B2 );
false
