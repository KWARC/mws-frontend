70.7-2 NrSubsTom
‣ NrSubsTom( tom )( attribute )
‣ OrdersTom( tom )( attribute )

Instead of storing the marks (see MarksTom (70.7-1)) of the table of marks tom one can use a matrix which contains at position ( i , j ) 𝑖 𝑗 (i,j) the number of subgroups of conjugacy class j 𝑗 j that are contained in one member of the conjugacy class i 𝑖 i . These values are stored in the NrSubsTom value in the same way as the marks in the MarksTom (70.7-1) value.

OrdersTom returns a list that contains at position i 𝑖 i the order of a representative of the i 𝑖 i -th conjugacy class of subgroups of tom.

One can compute the NrSubsTom and OrdersTom values from the MarksTom (70.7-1) value of tom and vice versa.

gap> NrSubsTom( a5 );
[ [ 1 ], [ 1, 1 ], [ 1, 1 ], [ 1, 3, 1 ], [ 1, 1 ], [ 1, 3, 1, 1 ], 
  [ 1, 5, 1, 1 ], [ 1, 3, 4, 1, 1 ], [ 1, 15, 10, 5, 6, 10, 6, 5, 1 ] 
 ]
gap> OrdersTom( a5 );
[ 1, 2, 3, 4, 5, 6, 10, 12, 60 ]
