56.5-8 IsPrime
‣ IsPrime( [R, ]r )( operation )

IsPrime returns true if the ring element r is a prime in the ring R, if given, and otherwise in its default ring (see DefaultRing (56.1-3)). If r is not a prime then false is returned.
