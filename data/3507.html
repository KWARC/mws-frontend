â€£ NestingDepthA( obj )( attribute )

For a GAP object obj, NestingDepthA returns the additive nesting depth of obj. This is defined recursively as the integer 0 0 if obj is not in IsGeneralizedRowVector (21.12-1), as the integer 1 1 1 if obj is an empty list in IsGeneralizedRowVector (21.12-1), and as 1 1 1 plus the additive nesting depth of the first bound entry in obj otherwise.
