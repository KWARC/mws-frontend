It has been replaced by the function IrreducibleSolvableGroupMS (50.11-1). Please note that the latter function does not guarantee any ordering of the groups in the database. However, for values of n, p, and i admissible to IrreducibleSolvableGroup, IrreducibleSolvableGroupMS (50.11-1) returns a representative of the same conjugacy class of subgroups of GL(n, p) as IrreducibleSolvableGroup did before.
