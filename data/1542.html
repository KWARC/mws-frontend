18.1-10 CoeffsCyc
‣ CoeffsCyc( cyc, N )( function )

Let cyc be a cyclotomic with conductor n 𝑛 n (see Conductor (18.1-7)). If N is not a multiple of n 𝑛 n then CoeffsCyc returns fail because cyc cannot be expressed in terms of N-th roots of unity. Otherwise CoeffsCyc returns a list of length N with entry at position j 𝑗 j equal to the coefficient of exp ⁡ ( 2 ⁢ π ⁢ i ⁢ ( j - 1 ) / 𝑁 ) 2 𝜋 𝑖 𝑗 1 N \exp(2\pi i(j-1)/\textit{N}) if this root belongs to the N-th Zumbroich basis (see 60.3), and equal to zero otherwise. So we have cyc = CoeffsCyc( cyc, N ) * List( [1..N], j -> E(N)^(j-1) ).
