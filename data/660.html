72.2-1 UnderlyingCharacterTable
‣ UnderlyingCharacterTable( psi )( attribute )

For a class function psi of the group G 𝐺 G , say, the character table of G 𝐺 G is stored as value of UnderlyingCharacterTable. The ordering of entries in the list psi (see ValuesOfClassFunction (72.2-2)) refers to the ordering of conjugacy classes in this character table.

If psi is an ordinary class function then the underlying character table is the ordinary character table of G 𝐺 G (see OrdinaryCharacterTable (71.8-4)), if psi is a class function in characteristic p ≠ 0 𝑝 0 p\neq 0 then the underlying character table is the p 𝑝 p -modular Brauer table of G 𝐺 G (see BrauerTable (71.3-2)). So the underlying characteristic of psi can be read off from the underlying character table.
