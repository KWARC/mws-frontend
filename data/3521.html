21.5-1 IsBound
‣ IsBound( list[, n] )( operation )
‣ IsBound( list[, ix1, ix2, ...] )( operation )

IsBound returns true if the list list has a element at index n, and false otherwise. list must evaluate to a list, otherwise an error is signalled.

gap> l := [ , 2, 3, , 5, , 7, , , , 11 ];;
gap> IsBound( l[7] );
true
gap> IsBound( l[4] );
false
gap> IsBound( l[101] );
false

IsBound(list[ix1,ix2,...] is a short-hand for IsBound(list[[ix1,ix2,...]]
