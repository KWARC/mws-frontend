70.5-2 PermutationTom
‣ PermutationTom( tom )( attribute )

For the table of marks tom of the group G 𝐺 G stored as UnderlyingGroup (70.7-7) value of tom, PermutationTom is a permutation π 𝜋 \pi such that the i 𝑖 i -th conjugacy class of subgroups of G 𝐺 G belongs to the i π superscript 𝑖 𝜋 i^{\pi} -th column and row of marks in tom.

This attribute value is bound only if tom was obtained from another table of marks by permuting with SortedTom (70.5-1), and there is no default method to compute its value.

The attribute is necessary because the original and the sorted table of marks have the same identifier and the same group, and information computed from the group may depend on the ordering of marks, for example the fusion from the ordinary character table of G 𝐺 G into tom.

gap> MarksTom( tom )[2];
[ 3, 3 ]
gap> MarksTom( sorted )[2];
[ 2, 2 ]
gap> HasPermutationTom( sorted );
true
gap> PermutationTom( sorted );
(2,3)
