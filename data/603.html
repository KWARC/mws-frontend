72.10-9 DnLatticeIterative
â€£ DnLatticeIterative( tbl, reducibles )( function )

Let tbl be an ordinary character table, and reducibles either a list of virtual characters of tbl or a record with components remainders and norms, for example a record returned by LLL (72.10-4).

DnLatticeIterative was designed for iterative use of DnLattice (72.10-8). DnLatticeIterative selects the vectors of norm 2 2 2 among the given virtual character, calls DnLattice (72.10-8) for them, reduces the virtual characters with found irreducibles, calls DnLattice (72.10-8) again for the remaining virtual characters, and so on, until no new irreducibles are found.

DnLatticeIterative returns a record with the same components and meaning of components as LLL (72.10-4).

>            [ 5, -1, 1, -1, 1 ], [ -1, 1, 3, -1, -1 ] ];;
gap> dn:= DnLatticeIterative( s4, red );
