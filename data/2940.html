39.2-2 GroupByGenerators
‣ GroupByGenerators( gens )( operation )
‣ GroupByGenerators( gens, id )( operation )

GroupByGenerators returns the group G 𝐺 G generated by the list gens. If a second argument id is present then this is stored as the identity element of the group.

The value of the attribute GeneratorsOfGroup (39.2-4) of G 𝐺 G need not be equal to gens. GroupByGenerators is the underlying operation called by Group (39.2-1).
