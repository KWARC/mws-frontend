64 Lie Algebras

A Lie algebra L 𝐿 L is an algebra such that x ⁢ x = 0 𝑥 𝑥 0 xx=0 and x ⁢ ( y ⁢ z ) + y ⁢ ( z ⁢ x ) + z ⁢ ( x ⁢ y ) = 0 𝑥 𝑦 𝑧 𝑦 𝑧 𝑥 𝑧 𝑥 𝑦 0 x(yz)+y(zx)+z(xy)=0 for all x , y , z ∈ L 𝑥 𝑦 𝑧 𝐿 x,y,z\in L . A common way of creating a Lie algebra is by taking an associative algebra together with the commutator as product. Therefore the product of two elements x , y 𝑥 𝑦 x,y of a Lie algebra is usually denoted by [ x , y ] 𝑥 𝑦 [x,y] , but in GAP this denotes the list of the elements x 𝑥 x and y 𝑦 y ; hence the product of elements is made by the usual *. This gives no problems when dealing with Lie algebras given by a table of structure constants. However, for matrix Lie algebras the situation is not so easy as * denotes the ordinary (associative) matrix multiplication. In GAP this problem is solved by wrapping elements of a matrix Lie algebra up as LieObjects, and then define the * for LieObjects to be the commutator (see 64.1).
