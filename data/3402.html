64.3-1 LieCentre
‣ LieCentre( L )( attribute )
‣ LieCenter( L )( attribute )

The Lie centre of the Lie algebra L is the kernel of the adjoint mapping, that is, the set { a ∈ L : ∀ x ∈ L : a ⁢ x = 0 } conditional-set 𝑎 𝐿 : for-all 𝑥 𝐿 𝑎 𝑥 0 \{a\in L:\forall x\in L:ax=0\} .

In characteristic 2 2 2 this may differ from the usual centre (that is the set of all a ∈ L 𝑎 𝐿 a\in L such that a ⁢ x = x ⁢ a 𝑎 𝑥 𝑥 𝑎 ax=xa for all x ∈ L 𝑥 𝐿 x\in L ). Therefore, this operation is named LieCentre and not Centre (35.4-5).

gap> L:= FullMatrixLieAlgebra( GF(3), 3 );
<Lie algebra over GF(3), with 5 generators>
gap> LieCentre( L );
<two-sided ideal in <Lie algebra of dimension 9 over GF(3)>, 
  (dimension 1)>
