18.1-7 Conductor
‣ Conductor( cyc )( attribute )
‣ Conductor( C )( attribute )

For an element cyc of a cyclotomic field, Conductor returns the smallest integer n 𝑛 n such that cyc is contained in the n 𝑛 n -th cyclotomic field. For a collection C of cyclotomics (for example a dense list of cyclotomics or a field of cyclotomics), Conductor returns the smallest integer n 𝑛 n such that all elements of C are contained in the n 𝑛 n -th cyclotomic field.

gap> Conductor( 0 ); Conductor( E(10) ); Conductor( E(12) );
1
5
12
