43.10-14 LargestElementStabChain
‣ LargestElementStabChain( S, id )( function )

Let G 𝐺 G be the group described by the stabilizer chain S. This function returns the element h ∈ G ℎ 𝐺 h\in G with the property that the list of images under h ℎ h of the base belonging to S is maximal w.r.t. lexicographical ordering. The second argument must be an identity element (used to start the recursion).
