24.6-3 RandomUnimodularMat
‣ RandomUnimodularMat( m )( function )

returns a new random mutable m × \times m matrix with integer entries that is invertible over the integers.

gap> m := RandomUnimodularMat(3);
[ [ -5, 1, 0 ], [ 12, -2, -1 ], [ -14, 3, 0 ] ]
gap> m^-1;
[ [ -3, 0, 1 ], [ -14, 0, 5 ], [ -8, -1, 2 ] ]
