39.2-3 GroupWithGenerators
‣ GroupWithGenerators( gens[, id] )( operation )

GroupWithGenerators returns the group G 𝐺 G generated by the list gens. If a second argument id is present then this is stored as the identity element of the group. The value of the attribute GeneratorsOfGroup (39.2-4) of G 𝐺 G is equal to gens.
