71.9-10 InverseClasses
‣ InverseClasses( tbl )( attribute )

For a character table tbl, InverseClasses returns the list mapping each conjugacy class to its inverse class. This list can be regarded as ( - 1 ) 1 (-1) -st power map of tbl (see PowerMap (73.1-1)).

gap> InverseClasses( CharacterTable( "A5" ) );
[ 1, 2, 3, 4, 5 ]
gap> InverseClasses( CharacterTable( "Cyclic", 3 ) );
[ 1, 3, 2 ]
