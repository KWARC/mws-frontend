71.20-4 CharacterTableIsoclinic
‣ CharacterTableIsoclinic( tbl[, classes][, centre] )( operation )
‣ SourceOfIsoclinicTable( tbl )( attribute )

If tbl is the (ordinary or modular) character table of a group with the structure 2 . G ⁢ .2 formulae-sequence 2 𝐺 .2 2.G.2 with a central subgroup Z 𝑍 Z of order 2 2 2 or 4 4 4 and a normal subgroup N 𝑁 N of index 2 2 2 that contains Z 𝑍 Z then CharacterTableIsoclinic returns the table of the isoclinic group in the sense of the Atlas of Finite Groups [CCNPW85, Chapter 6, Section 7]. If N 𝑁 N is not uniquely determined then the positions of the classes forming N 𝑁 N must be entered as list classes. If Z 𝑍 Z is not unique inside N 𝑁 N then the positions of the classes in Z 𝑍 Z must be entered as list centre; If Z 𝑍 Z has order 2 2 2 then centre can be also the position of the involution in Z 𝑍 Z .

Note that also if tbl is a Brauer table then classes and centre denote class numbers w.r.t. the ordinary character table.

For an ordinary character table that has been constructed via CharacterTableIsoclinic, the value of SourceOfIsoclinicTable is the list of three arguments in the CharacterTableIsoclinic call.

Note that there is no default method for computing the value of SourceOfIsoclinicTable.

gap> d8:= CharacterTable( "Dihedral", 8 );
CharacterTable( "Dihedral(8)" )
gap> nsg:= ClassPositionsOfNormalSubgroups( d8 );
[ [ 1 ], [ 1, 3 ], [ 1 .. 3 ], [ 1, 3, 4 ], [ 1, 3 .. 5 ], [ 1 .. 5 ] 
 ]
gap> iso:= CharacterTableIsoclinic( d8, nsg[3] );;
gap> Display( iso );
