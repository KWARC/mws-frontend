38.1-5 IsConfluent
‣ IsConfluent( rws )( property )
‣ IsConfluent( A )( property )

For a rewriting system rws, IsConfluent returns true if and only if rws is confluent. A rewriting system is confluent if, for every two words u 𝑢 u and v 𝑣 v in the free algebra T 𝑇 T which represent the same element of the algebra A 𝐴 A defined by rws, ReducedForm( rws, u 𝑢 u ) = ReducedForm( rws, v 𝑣 v ) as words in the free algebra T 𝑇 T . This element is the unique normal form of the element represented by u 𝑢 u .

For an algebra A with a canonical rewriting system associated with it, IsConfluent checks whether that rewriting system is confluent.

Also see IsConfluent (46.4-7).
