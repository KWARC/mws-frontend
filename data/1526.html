18.4-6 AtlasIrrationality
‣ AtlasIrrationality( irratname )( function )

Let irratname be a string that describes an irrational value as a linear combination in terms of the atomic irrationalities introduced in the sections 18.4-1, 18.4-2, 18.4-3, 18.4-4. These irrational values are defined in [CCNPW85, Chapter 6, Section 10], and the following description is mainly copied from there. If q N subscript 𝑞 𝑁 q_{N} is such a value (e.g. y 24 ′′ superscript subscript 𝑦 24 ′′ y_{24}^{\prime\prime} ) then linear combinations of algebraic conjugates of q N subscript 𝑞 𝑁 q_{N} are abbreviated as in the following examples:

2qN+3&5-4&7+&9means 2 ⁢ q N + 3 ⁢ q N * 5 - 4 ⁢ q N * 7 + q N * 9 2 subscript 𝑞 𝑁 3 superscript subscript 𝑞 𝑁 absent 5 4 superscript subscript 𝑞 𝑁 absent 7 superscript subscript 𝑞 𝑁 absent 9 2q_{N}+3q_{N}^{{*5}}-4q_{N}^{{*7}}+q_{N}^{{*9}}
4qN&3&5&7-3&4means 4 ⁢ ( q N + q N * 3 + q N * 5 + q N * 7 ) - 3 ⁢ q N * 11 4 subscript 𝑞 𝑁 superscript subscript 𝑞 𝑁 absent 3 superscript subscript 𝑞 𝑁 absent 5 superscript subscript 𝑞 𝑁 absent 7 3 superscript subscript 𝑞 𝑁 absent 11 4(q_{N}+q_{N}^{{*3}}+q_{N}^{{*5}}+q_{N}^{{*7}})-3q_{N}^{{*11}}
4qN*3&5+&7means 4 ⁢ ( q N * 3 + q N * 5 ) + q N * 7 4 superscript subscript 𝑞 𝑁 absent 3 superscript subscript 𝑞 𝑁 absent 5 superscript subscript 𝑞 𝑁 absent 7 4(q_{N}^{{*3}}+q_{N}^{{*5}})+q_{N}^{{*7}}

 


To explain the "ampersand" syntax in general we remark that "&k" is interpreted as q N * k superscript subscript 𝑞 𝑁 absent 𝑘 q_{N}^{{*k}} , where q N subscript 𝑞 𝑁 q_{N} is the most recently named atomic irrationality, and that the scope of any premultiplying coefficient is broken by a + + or - - sign, but not by & a ⁢ m ⁢ p ; 𝑎 𝑚 𝑝 \&amp; or * k absent 𝑘 *k . The algebraic conjugations indicated by the ampersands apply directly to the atomic irrationality q N subscript 𝑞 𝑁 q_{N} , even when, as in the last example, q N subscript 𝑞 𝑁 q_{N} first appears with another conjugacy * k absent 𝑘 *k .

E(7)^3+E(7)^5+E(7)^6
gap> AtlasIrrationality( "y'''24" );

gap> AtlasIrrationality( "-3y'''24*13&5" );
