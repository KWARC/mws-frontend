25.2-7 SmithNormalFormIntegerMatTransforms
‣ SmithNormalFormIntegerMatTransforms( mat )( operation )

This operation computes the Smith normal form of a matrix mat with integer entries. It returns a record with components normal (a matrix S 𝑆 S ), rowtrans (a matrix P 𝑃 P ), and coltrans (a matrix Q 𝑄 Q ) such that P ⁢ A ⁢ Q = S 𝑃 𝐴 𝑄 𝑆 PAQ=S .
