25.3-1 DeterminantIntMat
‣ DeterminantIntMat( mat )( function )

Computes the determinant of an integer matrix using the same strategy as NormalFormIntMat (25.2-9). This method is faster in general for matrices greater than 20 × 20 20 20 20\times 20 but quite a lot slower for smaller matrices. It therefore passes the work to the more general DeterminantMat (24.4-4) for these smaller matrices.
