For a group G with a generating set given in GeneratorsOfGroup (39.2-4), this function calculates the number of elements whose shortest expression as words in the generating set is of a particular length. It returns a list L, whose i + 1 ùëñ 1 i+1 entry counts the number of elements whose shortest word expression has length i ùëñ i . If a maximal length radius is given, only words up to length radius are counted. Otherwise the group must be finite and all elements are enumerated.
