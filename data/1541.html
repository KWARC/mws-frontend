18.1-11 DenominatorCyc
‣ DenominatorCyc( cyc )( function )

For a cyclotomic number cyc (see IsCyclotomic (18.1-3)), this function returns the smallest positive integer n 𝑛 n such that n 𝑛 n * cyc is a cyclotomic integer (see IsIntegralCyclotomic (18.1-4)). For rational numbers cyc, the result is the same as that of DenominatorRat (17.2-5).
