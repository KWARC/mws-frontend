41.8-3 AbelianSubfactorAction
‣ AbelianSubfactorAction( G, M, N )( operation )

Let G be a group and 𝑀 ≥ 𝑁 M N \textit{M}\geq\textit{N} be subgroups of a common parent that are normal under G, such that the subfactor 𝑀 / 𝑁 M N \textit{M}/\textit{N} is elementary abelian. The operation AbelianSubfactorAction returns a list [ phi, alpha, bas ] where bas is a list of elements of M which are representatives for a basis of 𝑀 / 𝑁 M N \textit{M}/\textit{N} , alpha is a map from M into a n 𝑛 n -dimensional row space over G ⁢ F ⁢ ( p ) 𝐺 𝐹 𝑝 GF(p) where [ 𝑀 : 𝑁 ] = p n fragments fragments [ M : N ] superscript 𝑝 𝑛 [\textit{M}:\textit{N}]=p^{n} that is the natural homomorphism of M by N with the quotient represented as an additive group. Finally phi is a homomorphism from G into G ⁢ L n ⁢ ( p ) 𝐺 subscript 𝐿 𝑛 𝑝 GL_{n}(p) that represents the action of G on the factor 𝑀 / 𝑁 M N \textit{M}/\textit{N} .

Note: If only matrices for the action are needed, LinearActionLayer (45.14-3) might be faster.
