50.11-5 PrimitiveIndexIrreducibleSolvableGroup
‣ PrimitiveIndexIrreducibleSolvableGroup( global variable )

This variable provides a way to get from irreducible solvable groups to primitive groups and vice versa. For the group G 𝐺 G = IrreducibleSolvableGroup( n, p, k ) and d = p n 𝑑 superscript 𝑝 𝑛 d=p^{n} , the entry PrimitiveIndexIrreducibleSolvableGroup[d][i] gives the index number of the semidirect product p n : G : superscript 𝑝 𝑛 𝐺 p^{n}:G in the library of primitive groups.

Searching for an index in this list with Position (21.16-1) gives the translation in the other direction.
