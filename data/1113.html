45.13-6 LGLength
‣ LGLength( G )( attribute )

returns the length of the LG-series of the group G, if G is solvable, and fail otherwise.

gap> G := SmallGroup( 96, 220 );
<pc group of size 96 with 6 generators>
gap> spec := SpecialPcgs( G );
Pcgs([ f1, f2, f3, f4, f5, f6 ])
gap> LGWeights(spec);
[ [ 1, 1, 2 ], [ 1, 1, 2 ], [ 1, 1, 2 ], [ 1, 1, 2 ], [ 1, 1, 3 ], 
  [ 1, 2, 2 ] ]
gap> LGLayers(spec);
[ 1, 1, 1, 1, 2, 3 ]
gap> LGFirst(spec);
[ 1, 5, 6, 7 ]
gap> LGLength( G );
3
gap> p := SpecialPcgs( Pcgs( SmallGroup( 96, 120 ) ) );
Pcgs([ f1, f2, f3, f4, f5, f6 ])
gap> LGWeights(p);
[ [ 1, 1, 2 ], [ 1, 1, 2 ], [ 1, 1, 2 ], [ 1, 2, 2 ], [ 1, 3, 2 ], 
  [ 2, 1, 3 ] ]

Thus the first group, SmallGroup(96, 220), has a lower nilpotent series of length 1 1 1 ; that is, the group is nilpotent. It is a direct product of its Sylow subgroups. Moreover the Sylow 2 2 2 -subgroup is generated by the elements f1, f2, f3, f4, f6, and the Sylow 3 3 3 -subgroup is generated by f5. The lower 2 2 2 -central series of the Sylow 2 2 2 -subgroup has length 2 2 2 and the second subgroup in this series is generated by f6.

The second group, SmallGroup(96, 120), has a lower nilpotent series of length 2 2 2 and hence is not nilpotent. The second subgroup in this series is just the Sylow 3 3 3 -subgroup and it is generated by f6. The subgroup generated by f1, … … \ldots , f5 is a Sylow 2 2 2 -subgroup of the group and also a head complement to the second head of the group. Its lower 2 2 2 -central series has length 2 2 2 .

In this example the FamilyPcgs (46.1-1) value of the groups used was a special pcgs, but this is not necessarily the case. For performance reasons it can be worth to enforce this, see IsomorphismSpecialPcGroup (46.5-3).
